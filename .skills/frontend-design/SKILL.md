# Frontend Design Skill

Aesthetic direction: DARK INDUSTRIAL / HACKER TERMINAL
- Background: deep charcoal (#0d0d0f) or near-black
- Accent: electric green (#00ff88) for found numbers, amber (#f59e0b) for comments
- Font pairing: "JetBrains Mono" (monospace, terminal feel) for data feeds, "Syne" or "DM Sans" for UI chrome
- Motion: staggered list item reveals as new comments/numbers arrive, subtle scanline texture overlay, glowing pulse on "LIVE" badge
- Layout: two-pane split on desktop, tabbed on mobile
- NO purple gradients. NO Inter font. NO rounded-2xl card-everywhere pattern.
